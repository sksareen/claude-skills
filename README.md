# sksareen-skills

A personal marketplace of Claude Code skills. Install individual skills into Claude Code via the plugin system.

## Install

```
/plugin marketplace add sksareen/claude-skills
/plugin install timeline-patterns@sksareen-skills
```

Then invoke with `/timeline-patterns`.

## Skills

### `timeline-patterns`

Opinionated UX pattern catalog for designing timeline, activity feed, history, and personal memory interfaces. Covers:

- **30+ named patterns** across five axes (time encoding, density, navigation, detail surfaces, search/filter)
- **Meta-patterns** specific to personal memory products (Three Questions model, Moment Card, Replay vs Explore)
- **7 ready-to-build combinations** composed from the patterns, each with effort/risk/fit called out
- **Anti-patterns** to avoid
- **Visual preview** (`previews.html`) rendering all seven combos as working mockups with sample data

Use when designing or prototyping any timeline-shaped interface: activity feeds, version history, dashboards, journaling tools, observability explorers, etc.
